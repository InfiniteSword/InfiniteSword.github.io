使用AI将文字转成svg

在我们做页面的时候，偶尔会有场景只有标题使用了一个艺术字体，此时如果引入一整个字体文件就会很不合适了，可能有的方案会选择使用图片来代替，不过图片会在某些情况下呈现锯齿且体积庞大。这时我们可以选择将文字转成svg图案，能同时兼顾显示质量与文件大小两个条件。

以我自己搭的某站为例：


将文字转换成svg的方法有以下至少两种。

用AI直接拿到文字轮廓

原料: Adobe Illustrator、你要转的字体
步骤:

  • 在AI中新建一个文档，用你要转的字体把字打在画布上。
  • 将字形转化为轮廓。用 Selection Tool 选中你写入的字，选择 Type –> Create Outlines 即可拿到这个字形的轮廓了。
  • 保存为svg格式就行了。

此方法学自：SVG文件：从Illustrator导文件到Web

只有文字的psd图

原料: Adobe Illustrator、Adobe Photoshop、文字的psd图
步骤:

  • 在PS中打开文字的psd图:

  • 选中文字的轮廓，这里我使用了颜色选区。选中后右键建立工作路径:

  • 导出路径到AI:
  • 用AI打开导出的.ai文件

  • 保存为svg格式，完成。

得到的svg可以根据自己的需要修改fill属性来改变填充颜色。
